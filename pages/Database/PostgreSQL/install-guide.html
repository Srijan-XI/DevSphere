PostgreSQL — Installation Guide

Install Postgres on Windows, macOS, and Linux. Create roles, databases, and verify with psql.

PSQL SERVICE SECURITY
Windows macOS Linux Verify
Prefer official installers or OS repositories for stable releases. For local dev, Docker is also convenient.

Windows

Option 1 — EDB Installer

# Download EDB installer and include: PostgreSQL Server, pgAdmin, psql

Option 2 — winget / Chocolatey

# winget
winget install PostgreSQL.PostgreSQL

# Chocolatey
choco install postgresql

macOS

# Homebrew
brew install postgresql@16
brew services start postgresql@16

# Use psql
psql --version

Linux

Ubuntu/Debian

sudo apt update
sudo apt install postgresql postgresql-contrib
sudo systemctl enable --now postgresql

# Switch to postgres user
sudo -u postgres psql

RHEL/CentOS/Fedora

sudo dnf install postgresql-server postgresql-contrib
sudo /usr/bin/postgresql-setup --initdb
sudo systemctl enable --now postgresql

Verify & Create Role/DB

# Connect to psql
psql -U postgres -h 127.0.0.1 -p 5432

# Inside psql
CREATE ROLE app LOGIN PASSWORD 'StrongP@ssw0rd';
CREATE DATABASE appdb OWNER app;
\c appdb
CREATE TABLE t(id BIGSERIAL PRIMARY KEY, name TEXT);
INSERT INTO t(name) VALUES ('Ada');
SELECT * FROM t;
\q
