Windows Installation

Option A: OpenJDK (Recommended)

  1. Download Eclipse Temurin (Adoptium) – free, TCK-certified OpenJDK.
  2. Run installer (.msi) and check "Add to PATH" and "Set JAVA_HOME".
  3. Choose LTS version (Java 21 or 17).

Manual PATH Setup (if needed):

# PowerShell (Admin)
[System.Environment]::SetEnvironmentVariable("JAVA_HOME", "C:\Program Files\Eclipse Adoptium\jdk-21.0.1.12-hotspot", "Machine")
[System.Environment]::SetEnvironmentVariable("Path", $env:Path + ";%JAVA_HOME%\bin", "Machine")

Option B: Oracle JDK

  1. Download from Oracle JDK Downloads.
  2. Run installer and follow prompts.
  3. Accept license terms (commercial use restrictions apply).

Option C: SDKMAN (WSL/Git Bash)

# Install SDKMAN
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"

# Install Java
sdk install java 21-tem
Note: Restart terminal after PATH changes to apply new settings.

macOS Installation

Option A: Homebrew (Easiest)

# Install Homebrew if missing
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install OpenJDK
brew install openjdk@21

# Symlink for system Java wrapper
sudo ln -sfn $(brew --prefix)/opt/openjdk@21/libexec/openjdk.jdk \
  /Library/Java/JavaVirtualMachines/openjdk-21.jdk

Option B: SDKMAN

curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install java 21-tem

Set JAVA_HOME (Add to ~/.zshrc or ~/.bash_profile)

export JAVA_HOME=$(/usr/libexec/java_home -v 21)
export PATH="$JAVA_HOME/bin:$PATH"

Reload shell: source ~/.zshrc

Linux Installation

Debian/Ubuntu

sudo apt update
sudo apt install openjdk-21-jdk -y

Fedora/RHEL

sudo dnf install java-21-openjdk-devel -y

Arch Linux

sudo pacman -S jdk-openjdk

SDKMAN (Universal)

curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install java 21-tem

Set JAVA_HOME (Add to ~/.bashrc)

export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export PATH="$JAVA_HOME/bin:$PATH"

Reload: source ~/.bashrc

Verification

java -version
javac -version
echo $JAVA_HOME  # Linux/macOS
echo %JAVA_HOME% # Windows CMD

Test Compilation

Create Test.java:

public class Test {
    public static void main(String[] args) {
        System.out.println("Java installed successfully!");
    }
}

Compile and run:

javac Test.java
java Test
Expected Output: Java installed successfully!

Build Tools

Maven

Install:

Windows: Download from maven.apache.org, extract, add bin to PATH.

macOS: brew install maven

Linux: sudo apt install maven (Debian/Ubuntu)

Verify:

mvn -version

Create Project:

mvn archetype:generate -DgroupId=com.example -DartifactId=myapp \
  -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
cd myapp
mvn package
java -cp target/myapp-1.0-SNAPSHOT.jar com.example.App

Gradle

Install:

Windows: Download from gradle.org, extract, add to PATH.

macOS: brew install gradle

Linux: sdk install gradle (via SDKMAN)

Verify:

gradle -version

Create Project:

gradle init --type java-application
gradle build
gradle run

IntelliJ IDEA Setup (Quick)

  1. Download IntelliJ IDEA Community (free).
  2. Install and launch.
  3. Create New Project → Java → Configure JDK (auto-detected or browse).
  4. Choose build system (Maven/Gradle) and dependencies.
Tip: IntelliJ auto-imports dependencies and provides intelligent code completion.

Best Practices

  • Use LTS versions: Java 21, 17, 11 for stability.
  • Manage versions: Use SDKMAN to switch between JDKs easily.
  • Build tools: Prefer Maven or Gradle over manual classpath management.
  • Code style: Follow Oracle's Java conventions or Google Java Style Guide.
  • Testing: Write unit tests with JUnit 5, integrate with CI/CD.
  • Dependencies: Use Maven Central or JCenter for libraries.
  • Security: Keep JDK updated for security patches.
Environment Variables Explained

JAVA_HOME: Points to JDK installation root (e.g., /usr/lib/jvm/java-21).

PATH: Includes $JAVA_HOME/bin to make java, javac accessible.

CLASSPATH: (Optional) Specifies where to find user classes. Build tools handle this automatically.

Verification Commands:

# Unix/macOS
echo $JAVA_HOME
echo $PATH | grep java

# Windows
echo %JAVA_HOME%
echo %PATH% | findstr java
Managing Multiple Java Versions

SDKMAN:

sdk list java
sdk install java 17-tem
sdk use java 17-tem
sdk default java 21-tem

macOS java_home:

/usr/libexec/java_home -V  # List installed JVMs
export JAVA_HOME=$(/usr/libexec/java_home -v 17)
