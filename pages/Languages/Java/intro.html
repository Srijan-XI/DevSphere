What Is Java?

Java is a class-based, object-oriented programming language designed for portability and reliability. Created by Sun Microsystems (now Oracle) in 1995, Java powers billions of devices worldwide – from Android apps to enterprise backends, big data platforms to microservices.

Key Characteristics

  • Object-Oriented: Everything is an object with classes, inheritance, and polymorphism.
  • Platform Independent: Compiled to bytecode, runs on any JVM.
  • Automatic Memory Management: Garbage collection handles cleanup.
  • Strongly Typed: Type safety enforced at compile time.
  • Rich Standard Library: Collections, networking, I/O, concurrency built-in.
  • Multithreading: Native support for concurrent programming.
Note: Java requires JDK (development) or JRE (runtime) installation. Modern versions include both in a single package.

Why Use Java?

Java dominates enterprise software, Android development, cloud services, and big data ecosystems. Its maturity, performance, and extensive tooling make it ideal for mission-critical applications.

Common Use Cases

Enterprise Apps Android Development Web Services Microservices Big Data (Hadoop) Cloud Platforms Scientific Computing Financial Systems E-commerce IoT Backends

Advantages

  • Portability: Write once, deploy anywhere with JVM.
  • Performance: JIT compilation optimizes runtime execution.
  • Ecosystem: Maven, Gradle, Spring, Hibernate, vast library ecosystem.
  • Community: Decades of expertise, extensive documentation.
  • Job Market: High demand in enterprise and Android sectors.
  • Security: Built-in security manager and bytecode verification.

Classes & Objects

public class Person {
    private String name;
    private int age;
    
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    public void introduce() {
        System.out.println("Hi, I'm " + name + ", " + age + " years old.");
    }
}

// Usage
Person p = new Person("Alice", 30);
p.introduce();

Exception Handling

try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.err.println("Cannot divide by zero!");
} finally {
    System.out.println("Cleanup always runs");
}

Mini Examples

File I/O

import java.nio.file.*;
import java.io.IOException;

try {
    String content = Files.readString(Path.of("data.txt"));
    System.out.println(content);
    
    Files.writeString(Path.of("output.txt"), "Hello World");
} catch (IOException e) {
    e.printStackTrace();
}

Streams API (Java 8+)

import java.util.stream.*;

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

int sum = numbers.stream()
    .filter(n -> n % 2 == 0)
    .mapToInt(Integer::intValue)
    .sum();

System.out.println("Sum of evens: " + sum);

Multithreading

class Worker extends Thread {
    public void run() {
        System.out.println("Worker thread running");
    }
}

// Usage
Worker w = new Worker();
w.start();
w.join(); // Wait for completion

Java Ecosystem

Build Tools

  • Maven: XML-based dependency management and build lifecycle.
  • Gradle: Groovy/Kotlin DSL, faster builds, better flexibility.
  • Ant: Legacy XML build tool.

Frameworks

  • Spring Boot: Microservices and web applications.
  • Hibernate: ORM (Object-Relational Mapping).
  • Apache Struts: MVC framework.
  • Quarkus: Cloud-native, Kubernetes-optimized Java.
  • Micronaut: Lightweight microservices framework.

Testing

  • JUnit: Unit testing standard.
  • Mockito: Mocking framework.
  • TestNG: Alternative testing framework.
