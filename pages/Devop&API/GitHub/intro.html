Repositories

GitHub hosts Git repositories with collaboration tools: code review, issues, discussions, releases, environments, and more.

# Create a repo on GitHub via CLI
gh repo create devsphere/demo --public --source=. --push

Branches & Pull Requests

# Create a feature branch
git checkout -b feat/docs
# Commit and push
git add . && git commit -m "docs: add overview"
git push -u origin feat/docs
# Open a PR
gh pr create --fill --base main

Issues & Projects

# Create an issue
gh issue create --title "Bug: crash on save" --body "Steps to reproduce..." --label bug

Use labels, milestones, and Projects for planning and tracking work.

GitHub Actions

# .github/workflows/ci.yml
name: CI
on: [push, pull_request]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with: { node-version: 20 }
      - run: npm ci && npm test

gh CLI Workflow

gh auth login
# Open PR in browser
gh pr view --web
# Merge PR (squash)
gh pr merge --squash
