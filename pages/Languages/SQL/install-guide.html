Note: SQL is a language, not a product. You'll need to install a database system (MySQL, PostgreSQL, SQLite, etc.) to practice SQL.

What to Install?

SQL (Structured Query Language) requires a database management system to execute queries. Choose one based on your needs:

  • SQLite: Lightweight, file-based, no server needed (best for learning)
  • MySQL: Popular, open-source, widely used in web applications
  • PostgreSQL: Advanced, feature-rich, excellent for complex queries
  • Microsoft SQL Server: Enterprise-grade, Windows-focused
  • MariaDB: MySQL fork with additional features
Recommendation for Beginners: Start with SQLite (no setup) or MySQL (most popular).

SQLite - Easiest Option

SQLite is perfect for learning SQL. No server setup required!

Windows

1 Download SQLite tools from: 
https://www.sqlite.org/download.html
2 Download "sqlite-tools-win32-x86-*.zip"
3 Extract and add the folder to your PATH

macOS/Linux

SQLite is usually pre-installed! Test with:

sqlite3 --version

If not installed:

# macOS (Homebrew)
brew install sqlite3

# Ubuntu/Debian
sudo apt install sqlite3

# Fedora
sudo dnf install sqlite

Quick Start

# Create/open a database
sqlite3 mydb.db

# Inside SQLite shell
sqlite> CREATE TABLE users (id INTEGER PRIMARY KEY, name TEXT);
sqlite> INSERT INTO users VALUES (1, 'Alice');
sqlite> SELECT * FROM users;
sqlite> .quit

MySQL Installation

Windows

1 Download MySQL Installer from: 
https://dev.mysql.com/downloads/installer/
2 Run the installer and choose "Developer Default"
3 Set a root password (remember it!)
4 Complete the installation wizard

macOS

# Using Homebrew (recommended)
brew install mysql

# Start MySQL service
brew services start mysql

# Secure installation
mysql_secure_installation

Linux (Ubuntu/Debian)

# Update package list
sudo apt update

# Install MySQL Server
sudo apt install mysql-server

# Start MySQL service
sudo systemctl start mysql

# Secure installation
sudo mysql_secure_installation

Connect to MySQL

# Connect as root
mysql -u root -p

# Inside MySQL
mysql> SHOW DATABASES;
mysql> CREATE DATABASE testdb;
mysql> USE testdb;
mysql> EXIT;

PostgreSQL Installation

Windows

1 Download from: 
https://www.postgresql.org/download/windows/
2 Run the installer and follow the wizard
3 Set a password for the postgres user

macOS

# Using Homebrew
brew install postgresql@15

# Start PostgreSQL
brew services start postgresql@15

# Or use Postgres.app (GUI application)
# Download from: https://postgresapp.com/

Linux (Ubuntu/Debian)

# Install PostgreSQL
sudo apt update
sudo apt install postgresql postgresql-contrib

# Start service
sudo systemctl start postgresql

# Switch to postgres user
sudo -i -u postgres
psql

Connect to PostgreSQL

# Connect to default database
psql -U postgres

# Inside psql
postgres=# \l              -- List databases
postgres=# CREATE DATABASE testdb;
postgres=# \c testdb       -- Connect to database
postgres=# \q              -- Quit

Verify Installation

SQLite

sqlite3 --version
# Expected: 3.x.x

MySQL

mysql --version
# Expected: mysql Ver 8.x.x

# Test connection
mysql -u root -p -e "SELECT VERSION();"

PostgreSQL

psql --version
# Expected: psql (PostgreSQL) 15.x

# Test connection
psql -U postgres -c "SELECT version();"
Success! If you can connect and run queries, your installation is complete.
