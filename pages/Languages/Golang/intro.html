What Is Go?

Go (often called Golang) is a statically typed, compiled language created at Google in 2009. Designed for simplicity, speed, and reliable concurrency, Go powers cloud infrastructure, microservices, DevOps tools, and distributed systems.

Key Characteristics

  • Fast Compilation: Compiles to native binaries in seconds.
  • Built-in Concurrency: Goroutines and channels for lightweight parallelism.
  • Simple Syntax: Clean, minimal design with 25 keywords.
  • Garbage Collection: Automatic memory management with low latency.
  • Standard Library: Comprehensive packages for HTTP, JSON, crypto, testing.
  • Cross-Compilation: Build for any platform from any platform.
Philosophy: Go favors simplicity over complexity ‚Äì no inheritance, generics are minimal, and error handling is explicit.

Why Use Go?

Go excels in building network services, cloud infrastructure, CLI tools, and concurrent systems. Its performance rivals C/C++ while maintaining developer productivity close to Python.

Common Use Cases

Microservices Cloud Infrastructure CLI Tools Web APIs DevOps Automation Distributed Systems Network Services Container Runtimes Database Engines Proxies & Load Balancers

Advantages

  • Speed: Compiled binaries run at near-C performance.
  • Concurrency: Lightweight goroutines scale to millions.
  • Simplicity: Easy to learn, read, and maintain.
  • Deployment: Single binary with no dependencies.
  • Tooling: Built-in formatter (gofmt), tester, profiler.
  • Community: Strong ecosystem for cloud-native development.
Examples: Docker, Kubernetes, Terraform, Prometheus, Consul all written in Go.

Goroutines & Channels

package main

import (
    "fmt"
    "time"
)

func worker(id int, ch chan string) {
    time.Sleep(time.Second)
    ch <- fmt.Sprintf("Worker %d done", id)
}

func main() {
    ch := make(chan string)
    
    for i := 1; i <= 3; i++ {
        go worker(i, ch)
    }
    
    for i := 1; i <= 3; i++ {
        msg := <-ch
        fmt.Println(msg)
    }
}

Mini Examples

HTTP Server

package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello from Go!")
}

func main() {
    http.HandleFunc("/", handler)
    fmt.Println("Server on :8080")
    http.ListenAndServe(":8080", nil)
}

JSON Encoding/Decoding

package main

import (
    "encoding/json"
    "fmt"
)

type User struct {
    Name  string `json:"name"`
    Email string `json:"email"`
}

func main() {
    u := User{Name: "Bob", Email: "bob@example.com"}
    
    // Encode
    data, _ := json.Marshal(u)
    fmt.Println(string(data))
    
    // Decode
    var u2 User
    json.Unmarshal(data, &u2)
    fmt.Printf("%+v\n", u2)
}

Error Handling

package main

import (
    "errors"
    "fmt"
)

func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

func main() {
    result, err := divide(10, 0)
    if err != nil {
        fmt.Println("Error:", err)
        return
    }
    fmt.Println("Result:", result)
}
Advanced Features

Interfaces: Implicit implementation via method sets.

type Writer interface {
    Write([]byte) (int, error)
}

// Any type with Write method satisfies Writer

Defer: Schedule function calls to run before return.

func readFile() {
    f, _ := os.Open("file.txt")
    defer f.Close() // Runs when function exits
    // ... work with f
}

Select: Multiplex channel operations.

select {
case msg := <-ch1:
    fmt.Println("From ch1:", msg)
case msg := <-ch2:
    fmt.Println("From ch2:", msg)
case <-time.After(time.Second):
    fmt.Println("Timeout")
}

Go Ecosystem

Popular Frameworks

  • Gin: High-performance HTTP web framework.
  • Echo: Minimalist web framework.
  • Fiber: Express-inspired framework.
  • Chi: Lightweight router.
  • gRPC-Go: Official gRPC implementation.

Database Drivers

  • database/sql: Standard database interface.
  • GORM: ORM library.
  • sqlx: Extensions to database/sql.
  • pgx: PostgreSQL driver and toolkit.

Tools

  • go fmt: Auto-formatter (standard style).
  • go test: Built-in testing framework.
  • go mod: Dependency management.
  • go build: Compile binaries.
  • golangci-lint: Meta-linter for code quality.
