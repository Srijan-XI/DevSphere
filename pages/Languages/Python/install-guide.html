Recommended: Install the latest Python 3.x version from python.org or use a package manager. Python 2.x is deprecated as of 2020.

Prerequisites

  • All Platforms: Admin/sudo privileges for system-wide installation
  • Windows: Windows 10 or later recommended
  • macOS: macOS 10.15+ (Python 3.x compatibility)
  • Linux: Most distributions come with Python pre-installed
  • Internet: For downloading Python and packages
Important: Always use Python 3.x (3.8+). Python 2.7 reached end-of-life in January 2020.

Windows Installation

Method 1: Official Installer (Recommended)

1 Download Python

Visit python.org/downloads and download the latest Python 3.x installer for Windows.

2 Run Installer

CRITICAL: Check "Add Python to PATH" before clicking "Install Now"

  • ‚úÖ Check "Add Python 3.x to PATH"
  • ‚úÖ Click "Install Now" (recommended) or "Customize installation"
3 Verify Installation

Open PowerShell or Command Prompt and verify:

python --version
# Output: Python 3.12.x

pip --version
# Output: pip 24.x.x from ...

Method 2: Microsoft Store (Windows 10/11)

1 Install from Microsoft Store

Open Microsoft Store and search for "Python 3.12" (or latest version). Click "Get" to install.

This method automatically adds Python to PATH and handles updates through Windows Update.

Method 3: Windows Package Manager (winget)

# Install latest Python
winget install Python.Python.3.12

# Or search for available versions
winget search Python.Python

Method 4: Chocolatey

# Install Python via Chocolatey
choco install python

# Install specific version
choco install python --version=3.12.0
PATH Issue? If python command not found, manually add to PATH:
  • Search "Environment Variables" in Windows
  • Edit "Path" under System Variables
  • Add: C:\Users\YourName\AppData\Local\Programs\Python\Python312
  • Add: C:\Users\YourName\AppData\Local\Programs\Python\Python312\Scripts

macOS Installation

Method 1: Official Installer (Recommended)

1 Download Python

Visit python.org/downloads/macos and download the macOS installer.

2 Install Package

Open the .pkg file and follow installation wizard. Default settings work for most users.

3 Verify Installation 
# macOS may have Python 2.7 as 'python'
# Use 'python3' for Python 3.x
python3 --version
# Output: Python 3.12.x

pip3 --version
# Output: pip 24.x.x

Method 2: Homebrew (Recommended for developers)

1 Install Homebrew (if not installed) 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2 Install Python 
brew install python@3.12

# Or install latest version
brew install python

# Update Python
brew upgrade python
3 Create Aliases (Optional) 
# Add to ~/.zshrc or ~/.bash_profile
echo 'alias python=python3' >> ~/.zshrc
echo 'alias pip=pip3' >> ~/.zshrc

# Reload shell
source ~/.zshrc

Method 3: pyenv (Version Manager)

# Install pyenv
brew install pyenv

# Add to shell configuration
echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc
echo 'command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.zshrc
echo 'eval "$(pyenv init -)"' >> ~/.zshrc

# Install Python version
pyenv install 3.12.0

# Set global version
pyenv global 3.12.0

Linux Installation

Check Existing Python

# Most Linux distros come with Python 3
python3 --version

# If not installed, proceed with installation below

Ubuntu/Debian

# Update package list
sudo apt update

# Install Python 3 and pip
sudo apt install python3 python3-pip python3-venv

# Install development headers (for compiling packages)
sudo apt install python3-dev build-essential

# Verify installation
python3 --version
pip3 --version

Fedora/RHEL/CentOS

# Install Python 3
sudo dnf install python3 python3-pip

# Or for older systems
sudo yum install python3 python3-pip

# Verify installation
python3 --version
pip3 --version

Arch Linux

# Install Python
sudo pacman -S python python-pip

# Verify installation
python --version
pip --version

openSUSE

# Install Python
sudo zypper install python3 python3-pip

# Verify installation
python3 --version
pip3 --version

Build from Source (Latest Version)

Install Python from Source 
# Install dependencies (Ubuntu/Debian)
sudo apt install build-essential zlib1g-dev libncurses5-dev \
  libgdbm-dev libnss3-dev libssl-dev libreadline-dev \
  libffi-dev libsqlite3-dev wget libbz2-dev

# Download Python
cd /tmp
wget https://www.python.org/ftp/python/3.12.0/Python-3.12.0.tgz

# Extract and build
tar -xf Python-3.12.0.tgz
cd Python-3.12.0
./configure --enable-optimizations
make -j $(nproc)

# Install
sudo make altinstall  # Use altinstall to not override system python

# Verify
python3.12 --version

Essential Tooling & Setup

pip - Package Manager

# Upgrade pip to latest version
python -m pip install --upgrade pip

# Install a package
pip install requests

# Install specific version
pip install django==4.2.7

# Install from requirements.txt
pip install -r requirements.txt

# Uninstall package
pip uninstall flask

# List installed packages
pip list

# Show package info
pip show numpy

# Search for packages
pip search data  # Note: Currently disabled on PyPI

Virtual Environments

Best Practice: Always use virtual environments for project isolation. This prevents package conflicts between projects. 
# Create virtual environment
python -m venv myenv

# Activate virtual environment

# Windows (PowerShell)
myenv\Scripts\Activate.ps1

# Windows (Command Prompt)
myenv\Scripts\activate.bat

# macOS/Linux
source myenv/bin/activate

# Install packages in venv
pip install django pandas numpy

# Save dependencies
pip freeze > requirements.txt

# Deactivate venv
deactivate

Alternative: virtualenv

# Install virtualenv
pip install virtualenv

# Create venv
virtualenv myenv

# Or specify Python version
virtualenv -p python3.11 myenv

IDE Setup - VS Code

Recommended Extensions:
  • Python (Microsoft): IntelliSense, linting, debugging, Jupyter support
  • Pylance: Fast, feature-rich language server
  • Python Indent: Correct Python indentation
  • autoDocstring: Generate docstrings automatically

Linting & Formatting

# Install tools
pip install pylint flake8 black autopep8

# Run pylint
pylint myfile.py

# Format with black
black myfile.py

# Auto-format with autopep8
autopep8 --in-place --aggressive myfile.py

# Flake8 style checker
flake8 myfile.py
