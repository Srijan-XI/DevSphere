Install Rust toolchain and set up your development environment for systems programming.

RUSTUP CARGO CROSS-PLATFORM
Prerequisites Windows macOS Linux Verify Tooling Next Steps
Recommended Method: Use rustup - the official Rust toolchain installer and version manager. It handles installation, updates, and managing multiple Rust versions.

Prerequisites

  • Windows: Visual Studio 2022 with C++ build tools or MinGW-w64
  • macOS: Xcode Command Line Tools (installs automatically)
  • Linux: GCC or Clang compiler and linker
  • All Platforms: Internet connection for downloading components
Important: Rust requires a C linker and sometimes C compiler. On Windows, you'll be prompted to install Visual Studio Build Tools if not present.

Windows Installation

Method 1: Rustup (Recommended)

1 Download Rustup Installer

Visit https://rustup.rs/ and download rustup-init.exe

2 Run Installer

Double-click the downloaded file. You'll be prompted to install Visual Studio C++ Build Tools if needed.

# Choose option 1 for default installation
# Or option 2 to customize installation
3 Install Visual Studio Build Tools (if prompted)

Follow the link provided by rustup to install Microsoft C++ Build Tools. Select "Desktop development with C++" workload.

4 Complete Installation

After VS Build Tools installation, re-run rustup-init.exe and select default installation.

5 Add to PATH

Rustup automatically adds Rust to PATH. Restart your terminal or PowerShell.

# Verify PATH (PowerShell)
$env:PATH -split ';' | Select-String cargo

Method 2: Windows Package Manager (winget)

# Install Rustup via winget
winget install Rustlang.Rustup

# Restart terminal and complete setup
rustup-init

Method 3: Chocolatey

# Install via Chocolatey
choco install rust

# Update Rust
rustup update

macOS Installation

Method 1: Rustup via Terminal (Recommended)

1 Install Xcode Command Line Tools (if not already installed) 
xcode-select --install
2 Install Rustup 
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
3 Follow Installation Prompts

Select option 1 for default installation. The installer will configure your PATH automatically.

4 Activate Rust Environment 
# Add to current shell
source "$HOME/.cargo/env"

# Or restart terminal

Method 2: Homebrew

# Install Rust via Homebrew
brew install rust

# Note: Homebrew installation doesn't include rustup
# For full toolchain management, use rustup method instead
Note: Homebrew installs Rust but not rustup. Using rustup (Method 1) is recommended for easier updates and toolchain management.

Linux Installation

Method 1: Rustup (Recommended for all distros)

1 Install Build Essentials 
# Ubuntu/Debian
sudo apt update
sudo apt install build-essential

# Fedora/RHEL/CentOS
sudo dnf install gcc gcc-c++

# Arch Linux
sudo pacman -S base-devel

# openSUSE
sudo zypper install -t pattern devel_basis
2 Install Rustup 
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
3 Select Installation Options

Choose option 1 for default installation. Rustup will modify your shell profile (~/.bashrc, ~/.zshrc, etc.)

4 Activate Environment 
# Add to current shell
source "$HOME/.cargo/env"

# Or restart terminal

Method 2: Package Managers

Ubuntu/Debian (apt) 
sudo apt update
sudo apt install rustc cargo

# Note: May not be latest version
# Use rustup for latest releases
Fedora/RHEL (dnf) 
sudo dnf install rust cargo
Arch Linux (pacman) 
sudo pacman -S rust

Verify Installation

# Check Rust compiler version
rustc --version
# Output: rustc 1.XX.X (hash YYYY-MM-DD)

# Check Cargo version
cargo --version
# Output: cargo 1.XX.X (hash YYYY-MM-DD)

# Check Rustup version
rustup --version
# Output: rustup 1.XX.X (hash YYYY-MM-DD)

Create and Run Hello World

# Create new project
cargo new hello_rust
cd hello_rust

# View generated code
cat src/main.rs

# Build and run
cargo run

# Expected output: Hello, world!

Essential Tooling & Setup

Update Rust

# Update Rust to latest stable
rustup update

# Check for updates
rustup check

# View installed toolchains
rustup show

Install Additional Components

# Install Rust formatter
rustup component add rustfmt

# Install Clippy (linter)
rustup component add clippy

# Install Rust Language Server (for IDE support)
rustup component add rust-analyzer

# View all components
rustup component list

Toolchain Management

# Install nightly toolchain
rustup toolchain install nightly

# Set default toolchain
rustup default stable

# Use nightly for specific project
cd my_project
rustup override set nightly

# Install specific version
rustup install 1.75.0

IDE Setup - VS Code

Recommended Extensions:
  • rust-analyzer: Official Rust language server with IntelliSense, refactoring, code completion
  • CodeLLDB: Native debugger support
  • Better TOML: Syntax highlighting for Cargo.toml
  • crates: Dependency version management
# Install rust-analyzer via rustup
rustup component add rust-analyzer

# VS Code will auto-detect and use it

Useful Cargo Commands

# Create new binary project
cargo new my_app

# Create new library
cargo new --lib my_lib

# Build project
cargo build

# Build with optimizations
cargo build --release

# Run project
cargo run

# Run with arguments
cargo run -- arg1 arg2

# Run tests
cargo test

# Check code without building
cargo check

# Format code
cargo fmt

# Run linter
cargo clippy

# Build documentation
cargo doc --open

# Add dependency
cargo add serde

# Update dependencies
cargo update

Install Popular Tools

# Install cargo-edit (for cargo add/rm/upgrade)
cargo install cargo-edit

# Install cargo-watch (auto-rebuild on changes)
cargo install cargo-watch

# Install cargo-expand (expand macros)
cargo install cargo-expand

# Install cargo-audit (security vulnerability scan)
cargo install cargo-audit

# Usage examples
cargo watch -x run          # Auto-run on file changes
cargo audit                 # Check for vulnerabilities
cargo upgrade               # Update dependencies in Cargo.toml
