What Are C & C++?

C is a procedural programming language created in the early 1970s, foundational to operating systems like UNIX, Linux, and countless embedded systems. It emphasizes simplicity, efficiency, and portability.

C++ extends C with object-oriented programming (OOP), generic programming via templates, and modern abstractions while maintaining backward compatibility. C++ powers game engines, databases, browsers, and high-performance applications.

Key Characteristics

  • Compiled: Direct to machine code for maximum speed.
  • Manual Memory Management: malloc/free (C), new/delete (C++).
  • Statically Typed: Type safety enforced at compile time.
  • Portable: Standards (C11, C17, C++17, C++20) ensure cross-platform code.
  • Low-Level Access: Pointers, inline assembly, hardware I/O.
Note: Manual memory management requires discipline—modern C++ emphasizes RAII and smart pointers to reduce errors.

Why Use C & C++?

C and C++ remain dominant in domains where performance, control, and efficiency are paramount. They power the infrastructure underlying modern computing.

C++: Classes & Objects

#include <iostream>
#include <string>

class Greeter {
    std::string name;
public:
    Greeter(std::string n) : name(n) {}
    void greet() {
        std::cout << "Hello, " << name << "!\n";
    }
};

int main() {
    Greeter g("DevSphere");
    g.greet();
    return 0;
}

Memory Management (C)

#include <stdlib.h>
#include <string.h>

int* allocate_array(int size) {
    int* arr = (int*)malloc(size * sizeof(int));
    if (arr == NULL) return NULL;
    memset(arr, 0, size * sizeof(int));
    return arr;
}

int main() {
    int* data = allocate_array(10);
    // Use data...
    free(data);
    return 0;
}

Smart Pointers (Modern C++)

#include <memory>
#include <vector>

int main() {
    auto data = std::make_unique<std::vector<int>>(10, 0);
    data->push_back(42);
    // Automatic cleanup, no manual delete
    return 0;
}

Mini Examples

File I/O (C)

#include <stdio.h>

int main() {
    FILE* fp = fopen("data.txt", "w");
    if (fp) {
        fprintf(fp, "Sample data\n");
        fclose(fp);
    }
    return 0;
}

STL Containers (C++)

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> nums = {5, 2, 8, 1, 9};
    std::sort(nums.begin(), nums.end());
    for (int n : nums) {
        std::cout << n << " ";
    }
    return 0;
}

Threading (C++11+)

#include <iostream>
#include <thread>
#include <chrono>

void worker(int id) {
    std::this_thread::sleep_for(std::chrono::seconds(1));
    std::cout << "Worker " << id << " done\n";
}

int main() {
    std::thread t1(worker, 1);
    std::thread t2(worker, 2);
    t1.join();
    t2.join();
    return 0;
}
Advanced Topics

Template Metaprogramming: Compile-time computation and type manipulation.

Move Semantics: C++11 rvalue references for efficient resource transfers.

RAII: Resource Acquisition Is Initialization—automatic cleanup pattern.

Constexpr: Compile-time evaluation of functions and variables.

Concepts (C++20): Constraints on template parameters for better error messages.

Modules (C++20): Modern replacement for header files.
