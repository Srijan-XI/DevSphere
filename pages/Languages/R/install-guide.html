Install R and RStudio for statistical computing and data visualization.

R 4.3+ RSTUDIO CRAN STATISTICAL COMPUTING
Prerequisites Windows macOS Linux RStudio Verify Next Steps
Recommended: Install both R and RStudio. R is the language, RStudio is the powerful IDE that makes working with R much easier.

Prerequisites

  • All Platforms: Admin/sudo privileges for system-wide installation
  • Windows: Windows 10 or later recommended
  • macOS: macOS 10.13+ (High Sierra or later)
  • Linux: Most distributions supported
  • Internet: For downloading R, RStudio, and packages
What You'll Install: R (the programming language) + RStudio (the IDE for R development)

Windows Installation

Step 1: Download R

1 Visit the CRAN (Comprehensive R Archive Network) website: 
https://cran.r-project.org/bin/windows/base/
2 Click "Download R-4.x.x for Windows" (latest version)
3 Run the downloaded .exe installer
4 Follow the installation wizard:
  • Select language
  • Accept license agreement
  • Choose installation directory (default is fine)
  • Select components (install all)
  • Choose startup options (default is fine)

Step 2: Add R to PATH (Optional but Recommended)

The installer usually adds R to PATH automatically. If not, add C:\Program Files\R\R-4.x.x\bin to your system PATH.

macOS Installation

Method 1: Download from CRAN (Recommended)

1 Visit the CRAN macOS page: 
https://cran.r-project.org/bin/macosx/
2 Download the appropriate .pkg file for your macOS version
3 Open the .pkg file and follow the installation instructions

Method 2: Using Homebrew

# Install Homebrew if not already installed
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install R
brew install r
Apple Silicon (M1/M2/M3): Make sure to download the ARM64 version for better performance.

Linux Installation

Ubuntu/Debian

# Update package list
sudo apt update

# Install R
sudo apt install r-base r-base-dev

# Install additional dependencies for packages
sudo apt install build-essential libcurl4-openssl-dev libssl-dev libxml2-dev

Fedora/RHEL/CentOS

# Install R
sudo dnf install R

# Or for older systems
sudo yum install R

Arch Linux

# Install R
sudo pacman -S r

Installing RStudio (Highly Recommended)

RStudio is the most popular IDE for R, providing a much better development experience than the basic R console.

Download RStudio Desktop

1 Visit the RStudio download page: 
https://posit.co/download/rstudio-desktop/
2 Download the appropriate installer for your OS:
  • Windows: .exe installer
  • macOS: .dmg file
  • Linux: .deb or .rpm package
3 Run the installer and follow the instructions
Note: RStudio requires R to be installed first. Make sure you've installed R before installing RStudio.

Verify Installation

Check R Version

Open a terminal/command prompt and run:

R --version

Expected output (version may vary):

R version 4.3.2 (2023-10-31) -- "Eye Holes"
Copyright (C) 2023 The R Foundation for Statistical Computing
Platform: x86_64-pc-linux-gnu (64-bit)

Run R Interactive Session

Type R to enter the R console:

R

You should see the R prompt >. Try a simple command:

> print("Hello from R!")
[1] "Hello from R!"

> 2 + 2
[1] 4

> q()  # Quit R

Test RStudio

Launch RStudio and verify you can see the console, editor, environment, and plots panels.

Success! If you can run R commands and see output, your installation is complete.

Installing R Packages

Install Packages from CRAN

R packages extend R's functionality. Install them using install.packages():

# Install a single package
install.packages("ggplot2")

# Install multiple packages
install.packages(c("dplyr", "tidyr", "readr"))

# Install with dependencies
install.packages("tidyverse", dependencies = TRUE)

Load Packages

# Load a package
library(ggplot2)

# Or
require(ggplot2)

Essential Packages for Beginners

# The Tidyverse (collection of data science packages)
install.packages("tidyverse")

# Data manipulation
install.packages("dplyr")

# Data visualization
install.packages("ggplot2")

# Reading data files
install.packages("readr")

# Working with dates
install.packages("lubridate")
