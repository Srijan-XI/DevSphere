Prerequisites

TypeScript requires Node.js and npm to be installed on your system.

Check if Node.js is Installed

node -v
npm -v

If not installed, visit the JavaScript Installation Guide to install Node.js first.

Recommended: Node.js LTS version (20.x or higher) for best compatibility.

Installing TypeScript

Global Installation

Install TypeScript globally to use the tsc compiler anywhere:

# Using npm
npm install -g typescript

# Using yarn
yarn global add typescript

# Using pnpm
pnpm add -g typescript

# Verify installation
tsc --version

Local Installation (Recommended for Projects)

Install TypeScript as a dev dependency in your project:

# Initialize package.json if needed
npm init -y

# Install TypeScript locally
npm install --save-dev typescript

# Verify (uses local version)
npx tsc --version
Best Practice: Use local installation for projects to ensure consistent TypeScript versions across team members.

TypeScript Configuration

Generate tsconfig.json

The tsconfig.json file configures TypeScript compiler options:

# Generate default tsconfig.json
npx tsc --init

Sample tsconfig.json

{
  "compilerOptions": {
    "target": "ES2020",              // Output JavaScript version
    "module": "commonjs",            // Module system
    "lib": ["ES2020"],               // Type definitions to include
    "outDir": "./dist",              // Output directory
    "rootDir": "./src",              // Source directory
    "strict": true,                  // Enable all strict type checks
    "esModuleInterop": true,         // Better CommonJS/ES module interop
    "skipLibCheck": true,            // Skip type checking of .d.ts files
    "forceConsistentCasingInFileNames": true,
    "resolveJsonModule": true,       // Import JSON files
    "declaration": true,             // Generate .d.ts files
    "sourceMap": true                // Generate source maps
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules", "dist"]
}

Important Compiler Options

Option Description
strict Enable all strict type checking (recommended)
target ECMAScript target version (ES5, ES2015, ES2020, etc.)
module Module system (commonjs, esnext, etc.)
outDir Output directory for compiled JavaScript
rootDir Root directory of TypeScript source files
noImplicitAny Error on expressions with implied 'any' type

Create Test File

Create hello.ts:

interface Person {
    name: string;
    age: number;
}

function greet(person: Person): string {
    return `Hello, ${person.name}! You are ${person.age} years old.`;
}

const user: Person = {
    name: "DevSphere",
    age: 5
};

console.log(greet(user));

Compile TypeScript

# Compile single file
tsc hello.ts

# This creates hello.js - run it:
node hello.js

# Compile all files in project
tsc

# Watch mode (auto-compile on changes)
tsc --watch
Expected Output: Hello, DevSphere! You are 5 years old.

Initialize New Project

# Create project directory
mkdir my-ts-project && cd my-ts-project

# Initialize npm
npm init -y

# Install TypeScript
npm install --save-dev typescript

# Generate tsconfig.json
npx tsc --init

# Create source directory
mkdir src

# Create main file
echo "console.log('Hello TypeScript!');" > src/index.ts

Add npm Scripts

Edit package.json:

{
  "scripts": {
    "build": "tsc",
    "watch": "tsc --watch",
    "start": "node dist/index.js",
    "dev": "tsc && node dist/index.js"
  }
}

Run Project

# Build TypeScript
npm run build

# Run compiled JavaScript
npm start

# Watch mode (auto-rebuild)
npm run watch

Development Tools

ts-node (Execute TypeScript Directly)

Run TypeScript files without compiling first:

# Install ts-node
npm install --save-dev ts-node @types/node

# Run TypeScript file directly
npx ts-node src/index.ts

# Add to package.json scripts
{
  "scripts": {
    "dev": "ts-node src/index.ts"
  }
}

tsx (Fast TypeScript Execution)

Modern, faster alternative to ts-node:

# Install tsx
npm install --save-dev tsx

# Run TypeScript
npx tsx src/index.ts

# Watch mode
npx tsx watch src/index.ts

Express + TypeScript

# Create project
mkdir my-express-ts && cd my-express-ts
npm init -y

# Install dependencies
npm install express
npm install --save-dev typescript @types/express @types/node ts-node

# Create tsconfig.json
npx tsc --init

# Create src/server.ts
mkdir src
cat > src/server.ts << 'EOF'
import express, { Request, Response } from 'express';

const app = express();
const PORT = 3000;

app.get('/', (req: Request, res: Response) => {
    res.send('Hello TypeScript!');
});

app.listen(PORT, () => {
    console.log(\`Server running on port \${PORT}\`);
});
EOF

# Run with ts-node
npx ts-node src/server.ts
