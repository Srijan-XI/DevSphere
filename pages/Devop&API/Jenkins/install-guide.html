Jenkins — Installation Guide

Run Jenkins with Docker or install natively on Linux. Complete the initial setup and create your first admin user.

DOCKER DEBIAN/UBUNTU INITIAL SETUP LTS
Docker Linux Initial Setup Verify

Run with Docker

docker volume create jenkins_home

docker run -d \
  --name jenkins \
  -p 8080:8080 -p 50000:50000 \
  -v jenkins_home:/var/jenkins_home \
  jenkins/jenkins:lts-jdk17

Open http://localhost:8080 after the container is healthy.

Install on Debian/Ubuntu

sudo apt update && sudo apt install -y openjdk-17-jdk curl gnupg
curl -fsSL https://pkg.jenkins.io/debian-stable/jenkins.io-2023.key | sudo tee \
  /usr/share/keyrings/jenkins-keyring.asc >/dev/null
echo deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] \
  https://pkg.jenkins.io/debian-stable binary/ | sudo tee \
  /etc/apt/sources.list.d/jenkins.list > /dev/null
sudo apt update && sudo apt install -y jenkins
sudo systemctl enable --now jenkins

Initial Setup

# Get admin password (Docker)
docker exec -it jenkins cat /var/jenkins_home/secrets/initialAdminPassword

# Native install
sudo cat /var/lib/jenkins/secrets/initialAdminPassword

Visit http://localhost:8080 → unlock using the password → install suggested plugins → create admin user → configure URL.

Verify

# Check service
systemctl status jenkins

# Open UI
http://localhost:8080
