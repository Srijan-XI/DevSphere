Git — Installation Guide

Install Git on Windows, macOS, and Linux. Configure your identity, editor, and SSH keys.

WINGET BREW APT SSH
Windows macOS Linux Config SSH Keys Verify

Windows

winget install -e --id Git.Git

Alternatively, download the installer from git-scm.com and follow prompts (use bundled Git Bash).

macOS

brew install git
# Or use Xcode Command Line Tools
xcode-select --install

Linux

# Debian/Ubuntu
sudo apt update && sudo apt install -y git

# Fedora
sudo dnf install -y git

Global Config

git config --global user.name "Your Name"
git config --global user.email "you@example.com"
# Optional defaults
git config --global init.defaultBranch main
git config --global core.editor "code --wait"

SSH Keys for GitHub

# Generate key
ssh-keygen -t ed25519 -C "you@example.com"

# Start agent and add key
ssh-agent -s
ssh-add ~/.ssh/id_ed25519

# Copy public key and add to GitHub settings
cat ~/.ssh/id_ed25519.pub

Verify

git --version
git config --list --show-origin
