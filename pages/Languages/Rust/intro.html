What Is Rust?

Rust is a modern systems programming language focused on safety, speed, and concurrency. Created by Mozilla Research and first released in 2010, Rust prevents common programming errors like null pointer dereferencing, buffer overflows, and data races through its innovative ownership system.

Key Characteristics

  • Memory Safety: No null pointers, dangling pointers, or data races.
  • Zero-Cost Abstractions: High-level features with no runtime overhead.
  • Ownership System: Unique compile-time memory management without GC.
  • Fearless Concurrency: Thread safety guaranteed by the type system.
  • Performance: C/C++ level performance with modern ergonomics.
  • Rich Type System: Expressive types, pattern matching, generics.
Note: Rust has a steeper learning curve than many languages, but the compiler's helpful error messages make learning easier.

Why Use Rust?

Rust excels in systems programming, web services, embedded systems, and anywhere performance and reliability are critical. It's been voted the "most loved programming language" in Stack Overflow surveys for 8 consecutive years.

Ownership System

fn main() {
    // Ownership rules:
    // 1. Each value has an owner
    // 2. There can only be one owner at a time
    // 3. When owner goes out of scope, value is dropped
    
    let s1 = String::from("hello");
    let s2 = s1;  // s1 is moved to s2, s1 no longer valid
    // println!("{}", s1);  // ‚ùå Error: value borrowed after move
    
    let s3 = String::from("world");
    let s4 = s3.clone();  // Deep copy
    println!("{} {}", s3, s4);  // ‚úÖ Both valid
    
    // References (borrowing)
    let s5 = String::from("hello");
    let len = calculate_length(&s5);  // Borrow s5
    println!("Length of '{}' is {}", s5, len);  // s5 still valid
}

fn calculate_length(s: &String) -> usize {
    s.len()
}

// Option type (no null in Rust!)
fn divide(a: f64, b: f64) -> Option {
    if b == 0.0 {
        None
    } else {
        Some(a / b)
    }
}

fn main() {
    match divide(10.0, 2.0) {
        Some(result) => println!("Result: {}", result),
        None => println!("Cannot divide by zero!"),
    }
}

Error Handling

use std::fs::File;
use std::io::ErrorKind;

fn main() {
    // Result type for recoverable errors
    let file_result = File::open("hello.txt");
    
    let file = match file_result {
        Ok(file) => file,
        Err(error) => match error.kind() {
            ErrorKind::NotFound => {
                println!("File not found!");
                return;
            }
            other_error => {
                panic!("Error opening file: {:?}", other_error);
            }
        },
    };
    
    // Using ? operator for error propagation
    fn read_username() -> Result {
        let mut username = String::new();
        File::open("username.txt")?.read_to_string(&mut username)?;
        Ok(username)
    }
}

Iterators & Closures

fn main() {
    // Iterators
    let v = vec![1, 2, 3, 4, 5];
    let sum: i32 = v.iter().sum();
    
    let doubled: Vec = v.iter()
        .map(|x| x * 2)
        .collect();
    
    let evens: Vec<&i32> = v.iter()
        .filter(|x| *x % 2 == 0)
        .collect();
    
    // Closures
    let add_one = |x| x + 1;
    println!("{}", add_one(5));  // 6
    
    let expensive_closure = |num| {
        println!("Calculating slowly...");
        std::thread::sleep(std::time::Duration::from_secs(2));
        num
    };
}
