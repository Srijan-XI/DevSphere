What Is PHP?

PHP (Hypertext Preprocessor) is a widely-used open-source server-side scripting language designed for web development. Created in 1995 by Rasmus Lerdorf, PHP can be embedded into HTML and is especially suited for creating dynamic web pages and applications.

Key Characteristics

  • Server-Side Execution: Code runs on the server, sending HTML to the client.
  • Easy to Learn: Simple syntax similar to C, Java, and Perl.
  • Cross-Platform: Runs on Windows, Linux, macOS, and Unix.
  • Database Integration: Native support for MySQL, PostgreSQL, SQLite, and more.
  • Flexible Typing: Dynamically typed with type hinting in modern versions.
  • Rich Ecosystem: Composer package manager, Laravel, Symfony frameworks.
Note: Modern PHP (8.0+) offers significant performance improvements, JIT compilation, and strong typing features.

Why Use PHP?

PHP remains a cornerstone of web development, powering content management systems, e-commerce platforms, and modern web applications. Its mature ecosystem and widespread hosting support make it ideal for rapid deployment.

Common Use Cases

Content Management E-commerce Web Applications REST APIs Blog Platforms Custom Websites Server Scripts Form Processing Session Management Database Apps

Advantages

  • Hosting Support: Available on virtually every web hosting provider.
  • Fast Development: Quick prototyping and rapid iteration.
  • Large Community: Millions of developers, extensive documentation.
  • Cost-Effective: Free, open-source, with affordable hosting options.
  • Framework Ecosystem: Laravel, Symfony, CodeIgniter, and more.
  • CMS Integration: Powers WordPress, Drupal, Joomla.

Database Connection (PDO)

<?php
try {
    $pdo = new PDO("mysql:host=localhost;dbname=mydb", "user", "pass");
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    
    $stmt = $pdo->prepare("SELECT * FROM users WHERE id = ?");
    $stmt->execute([1]);
    $user = $stmt->fetch(PDO::FETCH_ASSOC);
    
    echo $user["name"];
} catch (PDOException $e) {
    echo "Error: " . $e->getMessage();
}
?>
