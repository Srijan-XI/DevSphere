Spring Boot — Installation Guide

Install JDK + Maven/Gradle, generate a project with Spring Initializr, and run your app.

JDK 17+ Maven Gradle Initializr
Prerequisites Spring Initializr Maven Project Gradle Project Run Verify

Prerequisites

JDK 17 or 21

# Windows
winget install -e --id Oracle.JDK.17

# macOS
brew install openjdk@17

# Verify
java -version

Maven or Gradle

# Maven (Windows)
winget install -e --id Apache.Maven

# Maven (macOS)
brew install maven

# Gradle (Windows)
winget install -e --id Gradle.Gradle

# Gradle (macOS)
brew install gradle

# Verify
mvn -v
gradle -v

Spring Initializr

Generate a project from start.spring.io with dependencies like Web, JPA, PostgreSQL, etc.

# CLI alternative
curl https://start.spring.io/starter.zip \
  -d dependencies=web,data-jpa,postgresql \
  -d type=maven-project \
  -d javaVersion=17 \
  -d bootVersion=3.3.0 \
  -o demo.zip
unzip demo.zip
cd demo

Maven Project

# Build
mvn clean install

# Run
mvn spring-boot:run
<!-- pom.xml minimal -->
<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>3.3.0</version>
</parent>
<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>
</dependencies>

Gradle Project

# Build
./gradlew build

# Run
./gradlew bootRun
// build.gradle (Kotlin DSL)
plugins {
    id("org.springframework.boot") version "3.3.0"
    id("io.spring.dependency-management") version "1.1.4"
    kotlin("jvm") version "1.9.22"
}
dependencies {
    implementation("org.springframework.boot:spring-boot-starter-web")
}

Run

# Maven
mvn spring-boot:run

# Gradle
./gradlew bootRun

# Or via JAR
java -jar target/demo-0.0.1-SNAPSHOT.jar

Verify

curl http://localhost:8080
# Whitelabel Error Page (expected if no controllers)

# Add a controller to test
@RestController
class HelloController {
    @GetMapping("/")
    String hello() { return "Hello Spring Boot"; }
}
