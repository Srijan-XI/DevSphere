Browser JavaScript (No Installation Required)

JavaScript runs natively in every modern web browser. No installation needed for client-side development!

Getting Started

  1. Open any modern browser (Chrome, Firefox, Edge, Safari).
  2. Press F12 or right-click and select "Inspect".
  3. Navigate to the Console tab.
  4. Type JavaScript code directly and press Enter!

Quick Browser Test

// Open browser console (F12) and try:
console.log("Hello from the browser!");
alert("JavaScript is working!");

// Create elements dynamically
const div = document.createElement('div');
div.textContent = 'Created with JavaScript!';
document.body.appendChild(div);

Create HTML File

Save this as index.html and open in browser:

<!DOCTYPE html>
<html>
<head>
    <title>JavaScript Test</title>
</head>
<body>
    <h1>Hello World</h1>
    <button id="myBtn">Click Me</button>
    
    <script>
        document.getElementById('myBtn').addEventListener('click', () => {
            alert('JavaScript is working!');
        });
    </script>
</body>

Node.js (Server-Side JavaScript)

Node.js allows JavaScript to run outside the browser – on servers, build tools, command-line scripts, and more.

What Gets Installed

  • Node.js: JavaScript runtime environment
  • npm: Node Package Manager (world's largest package registry)
  • npx: Package runner tool (executes packages without installing)

Version Recommendations

Version Type Recommendation
LTS (Long Term Support) Even numbers (20.x, 22.x) ✅ Recommended for most users
Current Latest features ⚠️ For testing new features

Windows Installation

Option A: Official Installer (Recommended)

  1. Visit nodejs.org.
  2. Download the LTS version installer (.msi).
  3. Run installer and follow prompts (accept defaults).
  4. Check "Automatically install necessary tools" option.
  5. Restart terminal after installation.

Option B: Chocolatey

# Install Chocolatey first (if not installed)
Set-ExecutionPolicy Bypass -Scope Process -Force
iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

# Install Node.js
choco install nodejs-lts -y

# Verify
node -v
npm -v

Option C: nvm-windows (Version Manager)

  1. Download nvm-windows installer.
  2. Run nvm-setup.exe.
  3. Install and manage Node.js versions:
# List available versions
nvm list available

# Install specific version
nvm install 20.10.0

# Install LTS
nvm install lts

# Use specific version
nvm use 20.10.0

# Check current version
nvm current

macOS Installation

Option A: Homebrew (Recommended)

# Install Homebrew if missing
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install Node.js
brew install node

# Verify
node -v
npm -v

Option B: Official Installer

  1. Download macOS installer from nodejs.org.
  2. Run the .pkg file.
  3. Follow installation wizard.

Option C: nvm (Node Version Manager)

# Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash

# Reload shell configuration
source ~/.zshrc  # or ~/.bash_profile

# Install Node.js LTS
nvm install --lts

# Use installed version
nvm use --lts

# Set default version
nvm alias default node

Manage Multiple Node Versions

# List installed versions
nvm ls

# Install specific version
nvm install 18.19.0

# Switch between versions
nvm use 20
nvm use 18

Linux Installation

Debian/Ubuntu

# Using NodeSource repository (recommended)
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# Verify
node -v
npm -v

Fedora/RHEL/CentOS

# Using NodeSource
curl -fsSL https://rpm.nodesource.com/setup_lts.x | sudo bash -
sudo dnf install nodejs -y

Arch Linux

sudo pacman -S nodejs npm

nvm (Universal Linux)

# Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash

# Reload shell
source ~/.bashrc

# Install Node.js
nvm install --lts
nvm use --lts

Verification

node -v          # Check Node.js version
npm -v           # Check npm version
npx -v           # Check npx version

Package Managers

Tool Description Install Command
npm Default package manager (included with Node.js) npm install <package>
yarn Fast, reliable alternative to npm npm install -g yarn

Create a Project

npm init -y

# Install dependencies

npm install --save-dev nodemon jest eslint

# Start development server
npm run dev

Express.js (Backend)

mkdir my-server && cd my-server
npm init -y
npm install express

# Create server.js
echo "const express = require('express'); const app = express(); app.get('/', (req, res) => res.send('Hello!')); app.listen(3000);" > server.js

# Run server
node server.js

Best Practices

  • ✅ Use LTS version of Node.js for production projects.
  • ✅ Keep Node.js and npm updated regularly.
  • ✅ Use .gitignore to exclude node_modules from version control.
  • ✅ Lock dependencies with package-lock.json (npm) or yarn.lock.
  • ✅ Use environment variables for sensitive config (.env files).
  • ✅ Enable strict mode: "use strict";
  • ✅ Use ESLint and Prettier for consistent code quality.
  • ✅ Learn async/await instead of callback hell.
